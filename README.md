This project is a single-page, interactive quiz application developed using HTML, CSS, and JavaScript.
The quiz focuses on Computer Architecture concepts and allows users to test their knowledge through a clean and intuitive interface.
The entire application is built in one single HTML file, with embedded CSS and JavaScript, making it simple to run and easy to understand for beginners.

Objectives:
To understand JavaScript DOM manipulation
To build an interactive frontend application
To apply CSS styling and layout techniques
To create a task-oriented mini project suitable for academic submission

Features:
Start screen with Quiz Title & Start Button
5 Multiple Choice Questions (MCQs)
Option selection highlighted in light grey
“Next” button enabled only after selection
Automatic score calculation
Final result displayed at the center in bold
Clean UI with light purple background
Slightly rounded question container for better appearance
Fully responsive and user-friendly

Technologies Used:
HTML – Structure of the quiz
CSS – Styling and layout design
JavaScript – Quiz logic, interactivity, and score calculation
